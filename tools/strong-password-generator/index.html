Password Manager

Someday retinas will be scanned or some other biometric will make passwords a thing of the past. Until then we need passwords to get around. It should be common knowledge by now that having one password for multiple places is a terrible idea. One solution is to use a password manager. At it's core, a password manager is a place to store and retrieve you passwords securely. Many will argue on the pros and cons of one password manager over another. Regardless, using a password manager is better then no password manager.

Password Length

The longer the password, the more possible password combinations. Symbols might make a password look secure but the key is length. Longer passwords are favored over complex ones.
